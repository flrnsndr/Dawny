# Dawny — Marketing Website

Bilingual (DE/EN), story-driven one-pager for Dawny, built with **Astro 5 + Tailwind 4**.
Deploys as a static site to **IONOS Webhosting Plus** via FTPS.

## Quick start

```bash
# 1. install dependencies (use npm, pnpm, or bun)
npm install

# 2. run dev server (auto-generates QR + OG image first)
npm run dev
# → http://localhost:4321

# 3. build for production
npm run build
# → dist/

# 4. preview the production build locally
npm run preview
```

## Customising the brand

All design tokens are CSS variables defined in
[`src/styles/global.css`](src/styles/global.css) inside the `@theme {}` block.
Change colors, fonts, or container widths there and Tailwind picks them up.

Translations live in [`src/i18n/en.json`](src/i18n/en.json) and
[`src/i18n/de.json`](src/i18n/de.json). Sections automatically pull from the
matching dictionary based on the route's language prefix.

## Generating dynamic assets

`npm run dev` and `npm run build` both run `scripts/generate-qr.mjs` and
`scripts/generate-og.mjs` first:

- **QR code** points to the TestFlight invite (`/public/assets/qr-testflight.svg`).
  Edit the URL inside `scripts/generate-qr.mjs` if it ever changes.
- **OG image** is rendered from an inline SVG via `sharp` to
  `/public/og-image.png` (1200×630).

## Deploying to IONOS Webhosting Plus

IONOS Webhosting Plus serves static files (HTML/CSS/JS) directly. The
`/public/.htaccess` enables HTTPS-redirect, browser-language redirect at `/`,
and aggressive caching.

### Manual deploy (one-shot)

1. Copy `.env.example` → `.env` and fill in the SFTP credentials from the IONOS
   control panel (*Hosting → Webspace → SFTP/SSH access*).
2. Run:
   ```bash
   npm run build
   npm run deploy:ionos
   ```

### Automated deploy via GitHub Actions

The workflow at
[`../.github/workflows/deploy-website.yml`](../.github/workflows/deploy-website.yml)
builds and deploys on every push to `main` that touches `website/**`.

Add these repository secrets in **Settings → Secrets and variables → Actions**:

| Secret                 | Value                                            |
| ---------------------- | ------------------------------------------------ |
| `IONOS_FTP_HOST`       | e.g. `access-XXXXX.webspace-host.com`            |
| `IONOS_FTP_USER`       | e.g. `u00000000`                                 |
| `IONOS_FTP_PASSWORD`   | the SFTP/FTPS password                           |
| `IONOS_FTP_REMOTE_DIR` | `/` (or a subfolder if you host other sites too) |

## Performance & accessibility notes

- All animations respect `prefers-reduced-motion`.
- Fonts are self-hosted via `@fontsource-variable/*` — no Google Fonts CDN,
  GDPR-friendly.
- No cookies, no third-party scripts, no trackers.
- Lighthouse target: 95+ across all four categories.
